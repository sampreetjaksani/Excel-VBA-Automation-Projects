# Project 1 – Data Segregation Tool

## Overview

This project automates the process of separating a master dataset into multiple worksheets based on a predefined category. The workbook also includes user-friendly buttons to simplify common tasks.

## Features

* Segregation button to generate separate worksheets
* Delete Sheets button to remove generated worksheets while preserving the original workbook structure
* Simple interface for end users
* Reduces repetitive manual work in Excel

## Skills Demonstrated

* Excel VBA
* Worksheet Automation
* Button Automation
* Data Filtering
* Workbook Management
